`main.py` -- скрипт для получения части каталога интернет-магазина читай-город с последующим сохранением в .csv формате
`to_database` -- преобразование результатов выполнения `main.py` для загрузки в БД, результат сохраняется в нескольких .csv файлах
`chitai_gorod_db.sql` -- создание БД, заполнение БД полученными данными и примеры запросов
`data/` -- директория с готовыми к загрузке в БД файлами, после выполнения `main.py` и `to_database` от 06.08.2020
